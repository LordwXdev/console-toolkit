# Console Toolkit (C)

A simple console-based toolkit written in **C**.  
This project is made to practice core C fundamentals like input/output, loops, conditionals, and basic error handling.

------------------

## Features

- **Calculator**
  - Addition
  - Subtraction
  - Multiplication
  - Division (with division-by-zero check)

- **Menu-driven interface**
  - Runs in a loop until the user exits
  - Handles invalid inputs safely

-------------------

## Technologies Used

- Language: **C**
- Compiler: **GCC**
- Platform: Terminal / Command Line

-------------------

## How to Compile and Run

### Compile
```bash
gcc main.c -o toolkit
./toolkit
